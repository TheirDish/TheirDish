# TheirDish

사내 식당 메뉴를 편하게 보기 위해 만든 앱입니다.

Flutter 로 제작하였으며, 데이터는 인터넷에 공개되는 메뉴를 기반으로 합니다.

!!!!! 데이터를 가져오는 방식을 현재 차단한듯 합니다.
