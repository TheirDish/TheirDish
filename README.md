몇가지 UI 중심의 업데이트가 있었습니다. 주로 메뉴 공유 기능이나, 원하는 테이크아웃이나 서브 메뉴를 색칠해서 잘 보는 기능인데..
이거 설명 쓰기도 귀찮고 이것저것 만지다 보면 발견하시게 되리라 믿습니다.
애초에 이제 이걸 쓰시는 분이 얼마나 있는지도 모르겠습니다만..

의견은 Now Talk(DX부문)으로 주세요. 눈팅만 하고 답은 안달지만요..
